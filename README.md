# WaturiBet - Legit Sports Betting Platform

A modern sports betting interface built with HTML5, Tailwind CSS, and vanilla JavaScript.

## Features

- **Live Sports Betting** - Place bets on Premier League matches with real-time odds
- **Multiple Payment Methods** - M-Pesa, Bank Transfer, and Card payments
- **Deposit & Withdrawal** - Easy fund management with secure transactions
- **Live Odds** - Dynamic betting odds for various match outcomes
- **Dark Theme UI** - Modern dark interface optimized for user experience

## Getting Started

Simply open `index.html` in your web browser to start using WaturiBet.

### Requirements
- Modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for CDN resources (Tailwind CSS, Font Awesome)

## Usage

1. **Deposit Funds** - Click the "Deposit" button and enter your amount (KSh 10 - 100,000)
2. **Place a Bet** - Click on any match odds to place a bet
3. **Withdraw** - Use the "Withdraw" button to request funds (5% tax applied)

## Payment Options
- M-Pesa
- Bank Transfer
- Visa/Mastercard

## Withdrawal Details
- Minimum: KSh 50
- Maximum: KSh 300,000/day
- Tax: 5%
